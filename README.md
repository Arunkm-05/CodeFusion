# CodeFusion

Real-time collaborative coding platform with live code execution, chat, whiteboard, and room management.

## Features
- Live code sync across multiple users
- Python & JavaScript code execution
- Real-time chat with typing indicator
- Collaborative whiteboard
- JWT authentication
- Room passwords
- Auto-save, line numbers, auto-indent

## Tech Stack
- React.js, Socket.io, Node.js, Express, MongoDB, JWT
